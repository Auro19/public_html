JOOMLA ATTEND EVENTS COMPONENT

March 2007 :: Version 0.8.4

A Component that allows user registration for sessions/events.
A fork of Events Sessions 0.6.0 by Tony Blair
Supports optional integration with Community Builder Component and Events Component
