Registration Manager

The Registration Manager displays a list of registrations currently stored in the database. You can change the sorting order of this list (see Sorting), and/or isolate a limited subset of registrations (see Filtering). From this screen, the details of each registration can be viewed or exported, and a mass email message can be sent to a group of registrants (see Toolbar Actions).

Sorting

Sorting options are displayed using blue triangles in the header row (the currently selected option is orange). By default, registrations are sorted by ascending names. To sort by a different property, click the blue arrow next to the corresponding column's header text. The list will now be sorted by this column in ascending order. To switch to descending order, click the orange triangle again.

Filtering

For searching, enter a text fragment into the "Search" textbox and press Enter. Alternatively, you can display only registrations for a particular session by selecting this session from the pull-down menu.

Toolbar Actions

Most toolbar actions first require you to select a subset of registrations (to which the action will be applied) by clicking on the checkboxes on the far left of the screen. Shortcut: Click the single checkbox in the table header selects all sessions.

View
This will display the details of this particular registration. If multiple registrations are checked, only the first registration will be edited.
Delete
This will remove any information from the database pertaining to the currently selected registrations.
Email
An email message will be sent to the selected group of registrants. A content editing screen will be displayed for composing the message.
Export
The details of the selected registrations will be exported into a file for saving on the local machine.
