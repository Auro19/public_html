About

The About screen contains several tools for upgrading or uninstalling Attend Events.

Toolbar Actions

Uninstall
This will remove all Attend Events related information from your Joomla database. Afterwards, the page will redirect to the component installation page, so that you can uninstall the Attend Events component.
Settings
This will download a copy of your Attend Events configuration file. Replace the stock config.attend_events.php file included in the new version of Attend Events with this file (so that your current settings are saved in the upgrade process).
Update
If the database structure has changed between versions of Attend Events, this script will modify any Attend Events related tables in your Joomla database to the required structure of the new format.
