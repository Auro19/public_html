Configuration Manager

The Configuration Manager is used to control the behaviour of Attend Events, including the specification of default values for many session properties. For simplicity the configuration options have been grouped into four categories:

Options
Contains many controls for determining how Attend Events should operate. A default set of registration fields (when creating new sessions) can also be defined.
Thank-you Display
This provides a text editor for controlling what is displayed to the user after submitting a registration form.
Confirm Email
Controls whether the details of a registration should be emailed to the user as well. A text editor is also supplied for determining what should be sent in the message.
Notify Email
Controls whether a notification email should be sent to the session's host, informing them that a user has registered. A text editor is also supplied for determining what should be sent in the message.
