Session Manager

The Session Manager displays a list of sessions currently stored in the database. You can change the sorting order of this list (see Sorting), and/or isolate a limited subset of sessions (see Filtering). From this screen, new sessions may be created, and existing sessions may be edited or deleted (see Toolbar Actions). If integrated with the Events component, the name of the associated event (if any) will be displayed within parentheses following the session's name.

Sorting

Sorting options are displayed using blue triangles in the header row (the currently selected option is orange). By default, sessions are sorted by ascending names. To sort by a different property, click the blue arrow next to the corresponding column's header text. The list will now be sorted by this column in ascending order. To switch to descending order, click the orange triangle again.

Filtering

For searching, enter a text fragment into the "Search" textbox and press Enter. Alternatively, you can display only sessions which have a certain "status", "availability" or associated "event" (the Integrate with Events Component option in the Configuration :: Options screen must be set to true to enable this filtering ability).

Toolbar Actions

Most toolbar actions first require you to select a subset of sessions (to which the action will be applied) by clicking on the checkboxes on the far left of the screen. Shortcut: Click the single checkbox in the table header selects all sessions.

Publish
This will set the publish property of the selected sessions to true.
Unpublish
This will set the publish property of the selected sessions to false.
New
This will create a new session and open the Edit Session page.
Edit
This will open the Edit Session page for a particular session. If multiple sessions are checked, only the first session will be edited.
Delete
This will remove any information from the database pertaining to the currently selected sessions.
