Venue Details

The Venue Details page displays fields for specifying the location of the venue. Name and address are mandatory. If the Link to Google Maps configuration option is selected, the remaining fields should also be specified to ensure a successful search. If a URL is specified, a link to the venue's web page will also be included when displaying the location details in the front-end. Be sure to start the URL with http://
