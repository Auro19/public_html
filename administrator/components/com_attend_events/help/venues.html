Venues Manager

The Venue Manager displays a list of venues currently stored in the database. Venues can be associated with sessions, so that the location information of a frequently used venue doesn't have to be re-entered every time a new session is created. See the help screen of the Session Manager :: Edit Session screen for details on how to associate a venue with a particular session. For convenience, only a subset of venues can be displayed by filtering results (see Filtering). From this screen, new sessions may be created, and existing sessions may be edited or deleted (see Toolbar Actions).

Filtering

For searching, enter a text fragment into the "Search" textbox and press Enter.

Toolbar Actions

Most toolbar actions first require you to select a subset of venues (to which the action will be applied) by clicking on the checkboxes on the far left of the screen. Shortcut: Click the single checkbox in the table header selects all venues.

New
This will create a new session and open the Edit Venue page.
Edit
This will open the Edit Venue page for a particular session. If multiple sessions are checked, only the first venue will be edited.
Delete
This will remove any information from the database pertaining to the currently selected venues.
